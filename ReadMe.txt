This demo contain following functionality which can be helpful for newbie.


Note:
=====
- Change connectionString parameter for db credentials under root/protected/config/main.php

- YiiBooster extension downloaded under root/protected/extensions

- Language related folders under root/protected/messages

- Login Username/Password under root/protected/components/UserIdentity.php

- Database sql file under root/protected/data/HaveFunBase.sql


Functionality include:
======================

- Usage of Yii booster (menu, gridview, textbox, textarea usage).. ..
- Language conversion on dropdown change.
- File(image) upload, listing, tooltip in gridview when long text.
..
..
